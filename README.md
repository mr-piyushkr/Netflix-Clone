## ğŸš€ Project Overview

This **Netflix Clone** is a fully responsive **movie discovery and trailer streaming platform** that mimics the core experience of Netflix.  
It fetches **real-time movie data** from **TMDB** and allows users to browse through categories like **Trending, Netflix Originals, Top Rated, Action Movies**, and more.

Clicking on any movie opens its **official YouTube trailer**, giving users a true streaming-app feel.
